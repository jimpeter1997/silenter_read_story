### mysql   关系型数据库
- 一种特殊的存储数据的文件
- 查询速度极快
- 需要一套特殊的工具（RDBMS）来管理存储数据的文件



### RDBMS： Relational Database Management System

关系型数据库使用的一套软件： Mysql、Oracle、 sqlite、 Microsoft SQL Server

RDBMS-client ---> RDBMS-server ---> 特殊的存储数据的文件
mysql        ---> mysql-server ---> 真的存储的位置
